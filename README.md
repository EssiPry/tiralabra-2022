## Tietorakenteet ja algoritmit harjoitustyö

Ristinolla 25x25 ruudukolla. Peli on vielä kesken ja sitä voi tällä hetkellä pelata eri tyhmää bottia vastaan (hyvänmielen peli). Pelaaja on 0 ja botti on X.
